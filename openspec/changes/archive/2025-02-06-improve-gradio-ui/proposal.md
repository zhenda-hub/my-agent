## Why

当前 Gradio 界面较为简陋，缺乏现代化的 UI 设计和良好的用户体验。界面布局较为紧凑，视觉层次不清晰，交互方式不够直观，影响了用户的使用体验。

## What Changes

- 使用 Gradio 的现代主题和自定义 CSS 改进整体视觉效果
- 优化界面布局，增加视觉层次和间距
- 改进表单组件的样式（按钮、输入框、下拉菜单等）
- 添加更好的视觉反馈（加载状态、成功/错误提示）
- 优化聊天气泡样式，使其更易于阅读
- 改进响应式布局，支持不同屏幕尺寸
- 添加自定义主题配置，支持暗色/亮色模式切换（可选）

## Capabilities

### New Capabilities
- `modern-ui`: 现代化的用户界面设计，包括主题系统、自定义样式、响应式布局

### Modified Capabilities
- 无现有能力需要修改

## Impact

- **主要修改文件**: `src/web/app.py`
- **新增文件**: 可能需要添加 `src/web/themes.py` 或 `src/web/styles.py` 来管理主题和样式
- **依赖变化**: 无新增外部依赖（使用 Gradio 内置功能）
- **API 变化**: 无 API 变化，保持向后兼容
- **配置变化**: 可能需要添加 UI 相关配置项到 `src/config.py`
